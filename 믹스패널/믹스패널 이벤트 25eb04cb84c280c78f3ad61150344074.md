## 내용

- **상태** ( 관리하기 위한 이벤트 상태체크 )
- **이벤트명** (Mixpanel에 찍히는 정확한 문자열)
- **분류** ( 이벤트가 소속되는 카테고리 )
- **이벤트 정의** (이벤트가 어떤 행동을 의미하는지, 언제 찍혀야 하는지)
- **화면** ( 이벤트가 발생하는 정의 화면 설명 )
- **적용 (** web / app 적용 구분)
- **속성** (해당 이벤트와 연관된 속성값)

## 이벤트 네임 규칙

<aside>

**규칙 정의,순서**

`[Category]_[Action][Target][Context]`

- [Category] 이벤트 유형
- [Action] 행위
- [Target] 대상
- [Context] 발생위치/출처

**필수 구성 예시**

- PageView_Home
ㄴ`[Category]_[context]`
- EventOn_ClickPlayer
ㄴ`[Category]_[Action][Target]`
</aside>
